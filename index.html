How to Play

Write Python code to control your hero. Available commands:

  • hero.move_up(), hero.move_down(), hero.move_left(), hero.move_right() - Move the hero
  • hero.collect_gem() - Collect a gem if nearby
  • hero.find_nearest_gem() - Returns the nearest gem object
  • hero.say("message") - Display a message above the hero

After writing your code, click "Run Code" to execute it. Use "Reset" to restart the level.
